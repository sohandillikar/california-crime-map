## California Crime Map

### Prerequisites

npm comes bundled with Node.js. Install Node.js (which includes npm) from [nodejs.org](https://nodejs.org/). Verify installation:

```bash
node --version
npm --version
```

### Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser

### Tech Stack

- React 19 + TypeScript
- Vite
- Leaflet (maps)
- Recharts (charts)
- Tailwind CSS
